Remove Missing Tracks

Soft-delete tracks from the Rekordbox DB whose files no longer exist on disk.

⚠ Close Rekordbox before running.

This performs a soft delete (sets rb_local_deleted=1) — identical to how Rekordbox itself removes tracks. No rows are destroyed and a timestamped backup is created automatically before any changes.

How to restore a Rekordbox database backup

Every tool that modifies the database creates a timestamped backup before writing anything. Here's how to restore if something went wrong.

  1. Close Rekordbox completely before touching any database files.
  2. Open File Explorer and paste this into the address bar:
    %APPDATA%\Pioneer\rekordbox\
  3. You'll see master.db (live) and timestamped backups like master_backup_20250415_143022.db.
  4. Rename master.db → master_broken.db to keep it as a reference.
  5. Rename the backup you want to restore → master.db.
  6. Open Rekordbox — it loads from the restored file.

⚑ Delete old backups periodically to save disk space.
