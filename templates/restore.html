Restore Backup

How to recover your Rekordbox library if a tool run went wrong.

How backups work

Every tool that writes to the database creates a timestamped copy of master.db in the boxcutter-backups/ folder before the first change. Stream Rekordbox writes (ratings, tags, playlists) create one backup per session.

Backup filenames look like: master_backup_relocate_20250512_143022.db

How to restore — Windows
  1. Close Rekordbox completely before touching any database files.
  2. Open File Explorer and paste this into the address bar:
    %APPDATA%\Pioneer\rekordbox\boxcutter-backups\
  3. Find the timestamped backup you want to restore (newest is at the top when sorted by date).
  4. Copy it to %APPDATA%\Pioneer\rekordbox\ (one folder up).
  5. Rename the copy to master.db, replacing the current file.
  6. Open Rekordbox — it loads from the restored file.
How to restore — macOS
  1. Quit Rekordbox completely.
  2. Open Finder → Go → Go to Folder, then enter:
    ~/Library/Application Support/Pioneer/rekordbox/boxcutter-backups/
  3. Find the timestamped backup you want to restore.
  4. Copy it one folder up to the rekordbox/ folder.
  5. Rename the copy to master.db, replacing the current file.
  6. Open Rekordbox — it loads from the restored file.
Clean up old backups

Backups accumulate over time and can use significant disk space. Use Backup Cleaner to prune old copies, or delete them manually from the folder above.
