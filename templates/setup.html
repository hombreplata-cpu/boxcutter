Rekordbox Database
Path to your Rekordbox master.db file. Required — all DB tools and stats depend on it, and backups are stored next to this file.
Windows: Open File Explorer → paste %APPDATA%\Pioneer\rekordbox in the address bar → find master.db. If synced via Dropbox or OneDrive, check Rekordbox → Preferences → Advanced for the library path.
macOS: Default path is ~/Library/Application Support/Pioneer/rekordbox/master.db. If custom, check Rekordbox → Preferences → Advanced.
